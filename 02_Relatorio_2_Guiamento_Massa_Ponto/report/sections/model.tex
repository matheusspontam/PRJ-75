\section{Simulation Model}

The simulation was based on the mass-point Simulink model provided in class. The original file was copied and renamed to \texttt{MassaPontoMANSUP\_2018b.mdl}. The point-mass dynamics, propulsion table, drag block, seeker block, terminal guidance and airframe block were kept. The main changes were made in the pre-programmed guidance and in the switching logic between pre-programmed and terminal guidance.

The simulation is in the horizontal \(X\)-\(Z\) plane. Therefore, \(Z\) is lateral displacement, not altitude. The altitude of \(10~\text{m}\) is used only to initialize the atmosphere in \texttt{CondLanc.m}.

The pre-programmed block commands a lateral turn toward the known target bearing:

\begin{equation}
    a_{pre}=K_{pre}\left[\tan^{-1}\left(\frac{\Delta z}{\Delta x}\right)-\gamma\right]
\end{equation}

with saturation at \(5g\). After the switching condition is satisfied, the model uses the terminal proportional navigation block already present in the class model.
