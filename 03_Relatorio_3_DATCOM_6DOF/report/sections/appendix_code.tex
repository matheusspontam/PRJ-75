\section{Edited Code Excerpts}

The excerpt below shows the DATCOM generation script using the Exocet reference center and sweeping the control deflection:

\begin{lstlisting}[language=Matlab]
xcg = 2.700;
delta = -20:2:20;

write_exocet_for005(0, xcg);

out = fopen('coeficientes.dat', 'w');
fprintf(out, '%.6f ', xcg);
fprintf(out, '%d ', numel(delta));
fprintf(out, '%.8e ', delta);

for k = 1:numel(delta)
    write_exocet_for005(delta(k), xcg);
    status = system('Misdat.exe');
    ...
end

Coef2Mat('coeficientes.dat');
\end{lstlisting}

The same reference point was then imposed in the 6DOF mass/inertia file:

\begin{lstlisting}[language=Matlab]
% Same XCG used in DATCOM.
D.CRM = [2.700 0 0]';
\end{lstlisting}
